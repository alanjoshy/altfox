## Image Format
- **Format**: JPG (recommended for photos) or PNG (if transparency needed)
- **Optimization**: Compress images for web (aim for < 200KB per image)
- **Naming Convention**: Use lowercase with hyphens (e.g., `hero-carousel-business-analytics.jpg`)

## Image Quality Guidelines

1. **Resolution**: Use high-resolution source images (at least 2x the display size for retina displays)
2. **Compression**: Optimize for web - balance quality and file size
3. **Color**: Professional, consistent color grading across all images
4. **Style**: Modern, clean, professional aesthetic matching AloftX brand
5. **People**: If including people, ensure diverse representation
6. **Licensing**: Ensure all images have proper licensing for commercial use
